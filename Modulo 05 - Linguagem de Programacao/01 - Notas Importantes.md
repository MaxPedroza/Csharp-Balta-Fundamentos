# Notas importantes

---

## Objetivo

Antes de mergulhar no código, é importante alinhar algumas **boas práticas** e **expectativas** para o aprendizado de C#.

---

## Mentalidade para aprender programação

- **Pratique sempre**: programação se aprende fazendo.
- **Erros são normais**: mensagens de erro fazem parte do processo.
- **Comece simples**: entenda bem o básico antes de ir para assuntos avançados.
- **Experimente**: mude o código, teste de novo, veja o que acontece.

---

## Organização do código

- Use nomes **claros** para variáveis e métodos.
- Salve seus arquivos com frequência.
- Organize seus projetos em pastas quando começarem a crescer.

---

## Ferramentas

- Tenha o **.NET SDK** instalado.
- Use um editor apropriado (VS Code ou Visual Studio).
- Use o terminal/PowerShell para rodar comandos `dotnet`.

---

## Dica final

Mantenha este módulo como seu **guia de consulta rápida**. Volte aqui sempre que tiver dúvida sobre conceitos básicos de C#.
