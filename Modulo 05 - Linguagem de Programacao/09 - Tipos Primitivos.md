# Tipos primitivos

---

## O que são tipos primitivos?

São os **tipos básicos** da linguagem, usados para representar valores simples:

- Números inteiros.
- Números reais (com casas decimais).
- Caracteres.
- Valores lógicos (verdadeiro/falso).

---

## Principais tipos primitivos em C#

| Tipo  | Descrição                | Exemplo       |
|-------|--------------------------|---------------|
| `byte`  | inteiro de 0 a 255       | `byte b = 10;` |
| `int`   | inteiro                   | `int x = -5;`  |
| `long`  | inteiro longo             | `long l = 99;` |
| `float` | número real (32 bits)    | `float f = 1.5f;` |
| `double`| número real (64 bits)    | `double d = 2.7;` |
| `char`  | caractere único           | `char c = 'A';` |
| `bool`  | verdadeiro ou falso       | `bool ok = true;` |

---

## Escolhendo o tipo certo

- Use `int` para a maioria dos inteiros.
- Use `double` para números com casas decimais em geral.
- Use `bool` para condições lógicas.

Entender bem esses tipos é essencial para todos os próximos módulos.
