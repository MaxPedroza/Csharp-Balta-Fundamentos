# Operadores de atribuição

---

## Atribuição simples

```csharp
int x = 10;
```

---

## Operadores compostos

| Operador | Exemplo      | Equivalente a   |
|----------|--------------|-----------------|
| `+=`     | `x += 5;`    | `x = x + 5;`    |
| `-=`     | `x -= 3;`    | `x = x - 3;`    |
| `*=`     | `x *= 2;`    | `x = x * 2;`    |
| `/=`     | `x /= 4;`    | `x = x / 4;`    |
| `%=`     | `x %= 2;`    | `x = x % 2;`    |

---

## Exemplo

```csharp
int x = 10;
x += 5; // 15
x *= 2; // 30
```

Esses operadores deixam o código **mais conciso** e fácil de ler em muitos casos.
