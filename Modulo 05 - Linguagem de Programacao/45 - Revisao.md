# Revisão – Linguagem de Programação com C#

---

## O que você viu neste módulo

- Conceitos básicos: escopo, namespaces, `using`.
- Tipos fundamentais: variáveis, constantes, tipos primitivos, `string`, `char`, `bool`, `object`.
- Tipos especiais: `var`, tipos anuláveis, aliases, enums, structs.
- Conversões: implícita, explícita, `Parse`, `Convert`.
- Operadores: aritméticos, de atribuição, comparação e lógicos.
- Estruturas de controle: `if`, `switch`, laços `for`, `while`, `do/while`.
- Métodos e funções.
- Diferença entre **value types** e **reference types**.
