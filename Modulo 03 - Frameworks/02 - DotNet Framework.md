# .NET Framework

---

## Visão geral

O **.NET Framework** foi o primeiro grande framework da Microsoft para desenvolvimento de aplicações com C# (e outras linguagens .NET), lançado por volta de 2001.

Características principais:

- Focado em **Windows**.
- Ampla base instalada em desktops e servidores.
- Várias versões lançadas (1.0, 1.1, 2.0, 3.0, 3.5, 4.0, 4.5, 4.6, 4.7, 4.8...).
- Permite instalação **lado a lado** (Side by Side) de versões diferentes.

---

## Tipos de aplicações com .NET Framework

- Aplicações desktop (Windows Forms, WPF).
- Aplicações web antigas (ASP.NET "clássico").
- Serviços Windows.

Durante muitos anos, essa foi a base para a maioria dos sistemas corporativos em Windows.

---

## Limitações

Com o tempo, surgiram novos desafios:

- Necessidade de rodar em **Linux** e **macOS** (principalmente em nuvem).
- Demanda por melhor **performance** e **segurança**.
- Desejo de uma plataforma mais leve, modular e moderna.

O .NET Framework foi projetado inicialmente apenas para **Windows**, o que dificultava esses objetivos.

---

## Situação atual

- O .NET Framework ainda existe e é suportado, mas **não é recomendado para novos projetos**.
- A recomendação da Microsoft para aplicações novas é usar **.NET (Core/5+)**.

Você ainda vai encontrar muito código legado em .NET Framework no mercado, por isso é importante entender sua existência e contexto.
