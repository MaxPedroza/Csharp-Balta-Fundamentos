# Revisão – Frameworks

---

## O que você aprendeu neste módulo

1. **Frameworks**
   - Estrutura e conjunto de bibliotecas que aceleram o desenvolvimento.
2. **.NET Framework**
   - Focado em Windows, base de muitos sistemas legados.
3. **.NET Core**
   - Multiplataforma, moderno, open source, recomendado para novas aplicações.
4. **.NET Standard**
   - Conjunto de APIs em comum entre diferentes frameworks .NET.
5. **.NET 5**
   - Unificação da plataforma .NET, evolução natural do .NET Core.
6. **LTS**
   - Versões com suporte de longa duração, ideais para projetos estáveis.
7. **Versionamento**
   - Esquema MAIOR.MENOR.PATCH e conceito de breaking/minor changes.
8. **Runtime e SDK**
   - Diferença entre executar e desenvolver aplicações .NET.

---

## Mapa mental rápido

- Plataforma .NET
  - .NET Framework (legado, Windows).
  - .NET Core / .NET 5+ (moderno, multiplataforma).
- Padronização
  - .NET Standard para compatibilidade de bibliotecas.
- Ciclo de vida
  - Versionamento + versões LTS.
- Ferramentas
  - Runtime (executar) e SDK (desenvolver) + dotnet CLI.
