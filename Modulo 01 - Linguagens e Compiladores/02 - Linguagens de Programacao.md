# Linguagens de Programação

---

## O que é uma linguagem de programação?

Uma **linguagem de programação** é um conjunto de regras (sintaxe) e símbolos que permite que você escreva **instruções** para o computador.

Em resumo:

- Você escreve um **código-fonte** em uma linguagem (por exemplo, C#, Java, Python).
- Esse código é **traduzido** para algo que o computador entende (código de máquina).
- O resultado é um **programa** que executa tarefas reais.

---

## Componentes básicos de uma linguagem

Toda linguagem, independentemente do estilo, possui alguns elementos fundamentais:

- **Palavras-chave**: palavras reservadas da linguagem (`if`, `while`, `class`, `int`).
- **Tipos de dados**: números, textos, valores lógicos etc.
- **Operadores**: `+`, `-`, `*`, `/`, `==`, `&&`…
- **Estruturas de controle**: `if`, `else`, `switch`, laços de repetição.

Exemplo simples em C#:

```csharp
int idade = 25;

if (idade >= 18)
{
    Console.WriteLine("Você é maior de idade.");
}
```

## Paradigmas de programação

Um mesmo problema pode ser resolvido de formas diferentes, dependendo do **paradigma** da linguagem.

Principais paradigmas:

- **Imperativo / Procedural**: foco em comandos que alteram o estado do programa.
- **Orientado a Objetos (OO)**: foco em objetos que representam coisas do mundo real.
- **Funcional**: foco em funções e imutabilidade de dados.

Exemplos:

| Linguagem | Paradigma principal              | Uso comum                   |
|----------|----------------------------------|-----------------------------|
| C        | Procedural                       | Sistemas, drivers, embarcados|
| C#       | Orientado a Objetos / funcional  | Desktop, web, jogos         |
| Java     | Orientado a Objetos             | Backend, Android            |
| JavaScript | Multiparadigma                 | Web (front e back-end)      |

---

## Por que existem tantas linguagens?

Porque existem **problemas diferentes**, **plataformas diferentes** e **comunidades diferentes**.

Alguns fatores que influenciam a criação de novas linguagens:

- Performance.
- Produtividade do desenvolvedor.
- Facilidade de aprendizado.
- Plataforma alvo (web, mobile, desktop, embarcados…).

Ao longo da sua carreira, você provavelmente vai:

- Aprender **mais de uma** linguagem.
- Perceber que conceitos importantes **se repetem** entre elas.

Este módulo ajuda justamente a enxergar esses conceitos por trás das linguagens.
