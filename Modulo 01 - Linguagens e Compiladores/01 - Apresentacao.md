# Apresentação

---

## Objetivo da aula

Nesta aula vamos entender **por que** estudar linguagens de programação e compiladores é tão importante para quem está começando em programação.

Ao final, você será capaz de:

- Explicar, em poucas frases, o que é uma linguagem de programação.
- Dizer o que é um compilador em termos simples.
- Enxergar o caminho que o código faz até virar um programa executável.

---

## Motivação

Quando você escreve código, não está falando diretamente com o computador. Você escreve em uma **linguagem de programação**, e um **compilador** (ou interpretador) faz a ponte entre o que você escreveu e o que a máquina entende.

Pensar nisso ajuda a:

- Entender de onde vêm muitos erros de compilação.
- Escrever código mais consciente e organizado.
- Escolher melhor a linguagem certa para cada projeto.

---

## Visão geral do módulo

Neste módulo veremos, de forma progressiva:

1. O que são linguagens de programação e como são classificadas.
2. A diferença entre **alto** e **baixo nível**.
3. O que é uma linguagem **compilada** e uma **interpretada**.
4. Como funciona a **tipagem de dados**.
5. Uma **revisão geral** para fixar o conteúdo.

Use este material como um guia rápido enquanto assiste às aulas e quando for revisar antes de praticar com código real.
