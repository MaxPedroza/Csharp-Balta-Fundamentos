# Revisão – C#

---

## O que você viu neste módulo

1. **Por que utilizar C#?**
   - Linguagem moderna, forte no mercado, com grande ecossistema.
2. **C# como primeira linguagem**
   - Ótima para aprender lógica, tipos, OO e boas práticas.
3. **Código gerenciado**
   - Execução dentro do CLR, com serviços de segurança e gerenciamento.
4. **Compilação e gerenciamento**
   - Fluxo C# → IL → código de máquina, com JIT e Garbage Collector.
5. **IL**
   - Linguagem intermediária que torna o .NET multilíngue e multiplataforma.

---

## Mapa mental rápido

- C#
  - Orientado a Objetos, tipagem forte.
  - Roda sobre o .NET.
- Código gerenciado
  - Executado no CLR.
  - GC, segurança, gerenciamento de memória.
- Pipeline de execução
  - C# → compilação → IL → JIT → código nativo.
